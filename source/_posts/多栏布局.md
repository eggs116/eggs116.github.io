## 1. 左右结构

###  1.1 左固定，右自适应

#### 1.1.1浮动解决方案

```html
    <style>
        html,body{
            margin: 0;
            padding:0;
        }
        div{
            min-height: 100px;
        }
        .left {
            float: left;
            width: 300px;
            background: red;
        }
        .right {
             margin-left: 300px;
             background: yellowgreen;
        }
    </style>
    ```
    ```html
    <div class="wrap">
        <div class="left"></div>
        <div class="right">浮动解决方案 我宽度自适应我宽度自适应我宽度自适应
            <p>增加高度</p>
            <p>增加高度</p>
            <p>增加高度</p>
            <p>增加高度</p>
            <p>增加高度</p>
            <p>增加高度</p>
        </div>
    </div>
```

注：上面.right 的 margin-left 属性可以用 overflow:hidden 来替代（让.right 开启 BFC）
特点：两栏高度不一致
优点：兼容性好
缺点：浮动元素过多会导致页面结构复杂，并且要清除浮动，否则会导致页面混乱

```html

            <p>增加高度</p>
        </div>
    </div>
```

特点：两栏高度不一致
优点：兼容性好
缺点：脱离文档流，增加页面结构的复杂程度

#### 1.1.3 table 解决方案

```html
    <style>
        html,
        body {
            margin: 0;
            padding: 0;
        }

        div {
            min-height: 100px;
        }

        .wrap {
            display: table;
            width: 100%;
        }

        .left {
            display: table-cell;
            width: 300px;
            background: red;
        }

        .right {
            display: table-cell;
            background: yellowgreen;
        }
    </style>
```

```html

            <p>增加高度</p>
            <p>增加高度</p>
            <p>增加高度</p>

    </div>
```

特点：两栏高度一致

#### 1.1.5 grid 解决方案

```html
    <style>
        html,
        body {
            margin: 0;
            padding: 0;
        }

        div {
            min-height: 100px;
        }

        .wrap {
            display: grid;
            grid-template-columns: 300px auto;
        }

        .left {
            background: red;
        }

        .right {
            background: yellowgreen;
        }
    </style>
```
