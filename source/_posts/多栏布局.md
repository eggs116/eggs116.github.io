## 1 左右结构

### 1.1 左固定，右自适应

#### 1.1.1 浮动解决方案

````html
    <style>
        html,body{
            margin: 0;
            padding:0;
        }
        div{
            min-height: 100px;
        }
        .left {
            float: left;
            width: 300px;
            background: red;
        }
        .right {
             margin-left: 300px;
             background: yellowgreen;
        }
    </style>
    <div class="left"></div>
    <div class="right">浮动解决方案 我宽度自适应我宽度自适应我宽度自适应
        <p>增加高度</p>
        <p>增加高度</p>
        <p>增加高度</p>
        <p>增加高度</p>
        <p>增加高度</p>
        <p>增加高度</p>
    </div>
```
注：上面.right的margin-left属性可以用overflow:hidden来替代（让.right开启BFC）
优点：兼容性好
缺点：浮动元素过多会导致页面结构复杂，并且要清除浮动，否则会导致页面混乱
